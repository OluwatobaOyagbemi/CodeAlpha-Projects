🔍 Key Findings
😷🦠 COVID-19 Impact
Unemployment was stable before COVID
During lockdown there was a → sharp spike📈
Some months show very high values (15–35%), far above normal levels📶
After restrictions eased → gradual recovery
Conclusion:

COVID caused a sudden shock, not a slow increase.

📅 Seasonal Trends
Small monthly fluctuations normally
Rural/agriculture jobs might have shown seasonal effects
Festive/harvest months → lower unemployment
Lockdown months break this pattern
Conclusion:

Seasonality exists, but crises dominate trends.

🏙️ Regional Patterns
